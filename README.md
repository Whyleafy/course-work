## Назначение

Приложение предназначено для демонстрации принципов построения прикладных desktop‑систем:

* работа с реляционной БД;
* документооборот (черновик → проведение → отмена);
* учёт складских движений;
* расчёт остатков;
* использование Qt Widgets и Qt SQL.

---

## Основные сущности

* товары;
* контрагенты;
* документы;
* строки документов;
* складские движения;
* остатки товаров.

Документы имеют статусы:

* DRAFT — черновик;
* POSTED — проведён;
* CANCELLED — отменён.

---

## Требования

* CMake 3.16 и выше
* Qt 6 (Widgets, SQL)
* компилятор с поддержкой C++17

Поддерживаемые ОС:

* Windows
* macOS
* Linux

---

## Сборка

Перед сборкой необходимо установить Qt и указать путь к нему через переменную окружения `CMAKE_PREFIX_PATH`.

Пример:

```
C:\Qt\6.x.x\msvc2019_64
~/Qt/6.x.x/macos
~/Qt/6.x.x/gcc_64
```

Сборка проекта:

```bash
cmake -S . -B build-release -DCMAKE_BUILD_TYPE=Release
cmake --build build-release -j
```

---

## Запуск

Windows:

```
build-release\WholesaleTradeApp.exe
```

macOS:

```
open build-release/WholesaleTradeApp.app
```

Linux:

```
./build-release/WholesaleTradeApp
```
